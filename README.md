# VSCode Markdown Automatic Bullet Points

An open-source "mini-extension" which automatically adds bullet points when editing in Markdown.

## Features

- Automatically inserts bullet point when starting new line
- Detects the type of bullet point and spacing
- Turns off the type event when not in markdown to reduce resource use

## Release Notes

### 1.0.0

Initial release

## Known Issues

- After changing the document language to markdown, the user must type at least one character before the event rewires itself. If the first key pressed is Enter, the extension will not add a bullet point. This should not be an issue under normal circumstances.
